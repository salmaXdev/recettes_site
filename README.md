e projet consiste à créer un site web dédié aux recettes de cuisine. Le site présentera 5 recettes différentes, avec pour chacune une description, les ingrédients nécessaires ainsi que les étapes de préparation.

L’objectif de ce projet est de concevoir une interface simple, claire et agréable, afin de permettre aux utilisateurs de consulter facilement des recettes et de s’inspirer en cuisine
